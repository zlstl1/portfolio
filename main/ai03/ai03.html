딥러닝기반 인공지능 기술을 이용한 95%이상 성능을 가지는 영상분석기술

과제요약

과제 기간 : 2019-09-26 ~ 2020-06-25

Github : https://github.com/zlstl1/ACAM

주요기술 : Python, Pytorch, PyQT

주요기능 : CCTV 관제사 부족으로 인한 관제사각 해소를 위하여 딥러닝 기반의 객체탐지, 객체추적, 행위추론, 이상상황 감지 등의 영상분석기술 제공

수행업무

PyQT를 이용하여 GUI를 구성하며 NAS와 연동하여 영상 Load 기능 구현. 영상에 대한 Player 기능을 구현하고 TF object를 통해 객체 탐지 기능 구현
ALPR 기반 차량 번호판 탐지 및 인식 기능 구현. DeepSort 알고리즘을 통해 객체 추적을 구현하고 ACAM 모델 기반으로 행위 추론 기능 구현. 특정 탐지에 대한 북마크 기능 및 이상상황(낙상, 월담 등)에 대한 알림 기능 구현
서버 자원 모니터링을 위하여 Prometheus를 연결하여 자원 사용량 모니터링 페이지 제작
담당업무 : 자동차 번호판 식별, 객체 인식 및 추적 모델 개발, UI 제작, TTA인증

각각의 성능은 TTA 인증을 받았으며 ①자동차 번호판 인식률 : 100%, ②다중 객체 검출률 : 100%, ③객체 추적률 : 100%, ④객체(사람) 모션 인식률 : 100%, ⑤책갈피 기능 정확도 : 100%, ⑥이상 상황 알람 정확도 : 100% 달성
※사용된 시료는 부산시에서 실제 CCTV를 통해 검출이 잘 되도록 연출된 영상을 활용하였음.
