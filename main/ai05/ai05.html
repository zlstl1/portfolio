교통 약자 주행 영상 AI데이터 구축

과제요약

과제 기간 : 2020.09.16 ~ 2020.12.31

Github : https://github.com/zlstl1/yolov5_custom
https://github.com/zlstl1/yolov5_server

주요기술 : Python, Pytorch, Labelme, Spring

주요기능 : 교통 약자들의 이동 편의성 향상을 위한 데이터 구축 및 활용 서비스 제공

수행업무

교통 약자들의 경우 법적으로 차도로 다니지 못하나 인도의 노면 상태로 인하여 불가피하게 차도를 이용하여 사고가 발생하는 경우가 많으며 이를 인공지능 기반의 노면, 공간 정보 분석을 통하여 사전에 보수공사를 수행하기 위한 데이터 구축 사업 수행
데이터 수집용 앱 개발을 통하여 노면 및 공간 영상을 촬영하며 해당 공간의 진동 값을 함께 수집하고 이에 대한 데이터의 정제, 가공, 모델 개발
노면의 경우 국토교통부에서 제공하는 노면 등급 기준에 따라 A~E 등급으로 분류하며 학습모델을 통해 새로운 노면의 등급을 분류하여 보수가 필요한 노면의 경우 알림을 발생하는 활용 서비스 개발
담당업무 : 데이터 가공툴 개발(오픈소스 커스터마이징), 노면 및 공간 분류 학습 모델 개발
