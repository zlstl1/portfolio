신종 감염병 해외유입 예측 및 지능적 차단 기술개발

과제요약

과제 기간 : 2021.04.01 ~ 2023.12.31

Github : https://github.com/zlstl1/thermalcamera

주요기술 : Android, Python, Pytorch, JAVA

주요기능 : COVID-19와 같은 신종감염병의 선제적 대응을 위하여 고발열자, 마스크 미/오착용자 탐지 기능 제공

수행업무

행정안전부 주관의 사업으로 COVID-19와 같은 신종감염병의 선제적 대응을 위한 시스템 개발 사업 수행
총 6개의 기관이 협업하여 100여개국 웹 크롤링, 크롤링 데이터 NLP를 통한 감염병 위험도 분석, 방역 터널 및 수화물 소독 장치 개발, 무인 민원 로봇 개발, 인공지능 기반 열화상 카메라 모듈 개발을 각각 수행하였으며 저희 기관은 인공지능 기반 열화상 카메라 모듈 개발 수행
열화상 카메라 모듈은 RGB/IR 이미지를 촬영하는 열화상 카메라와 이미지를 전처리하는 Android 보드를 결합하여 수집된 이미지를 서버 PC로 전달하고 서버 PC는 이를 인공지능 모델을 통해 안면검출, 안면 영역의 온도값 산정, 마스크 착용/오착용/미착용 검출을 수행하여 고발열자 또는 마스크 오착용/미착용자 발생 시 모니터링 페이지로 이벤트 전달
담당업무 : 열화상 카메라 모듈의 Android 앱 개발, 안면 검출/마스크 검출 모델 학습
