GPU HPC 기반의 360도 영상 스티칭 도구 및 클라우드 서비스 개발

과제요약

과제 기간 : 2018-12-01 ~ 2019-11-30

주요기술 : Spring Framework, Prometheus, FFMPEG

주요기능 : 다수의 드론 또는 CCTV로부터 촬영된 영상을 파노라마형태로 연결하여 하나의 영상으로 이어 붙이는 기능 제공

수행업무

웹 클라우드의 형태로 파일을 업/다운로드가 가능하도록 Spring을 활용하여 사용자 권한관리, 작업 로그관리 등 페이지 제작업무를 수행함. 또한 업로드된 영상을 FFMPEG을 통해 Framerate 변환, 영상 스티칭 등의 영상 처리 수행
서버 자원 모니터링을 위하여 Prometheus를 연결하여 자원 사용량 모니터링 페이지 제작. 변환 기능은 REST API를 통해 작업이 가능하도록 서버 구축
담당업무 : 스티칭 웹페이지 제작(RestAPI, 웹 UI)
