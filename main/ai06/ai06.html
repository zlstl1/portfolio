유동 인구 분석을 위한 CCTV 영상 데이터 구축

과제요약

과제 기간 : 2021.07.01 ~ 2021.12.31

Github : https://github.com/zlstl1/Foottraffic-processing
https://github.com/zlstl1/Hackerton

주요기술 : Python, Pytorch, Labelme

주요기능 : 도심지의 유동인구 분석을 위하여 영상 수집, 정제, 가공을 통한 데이터 구축 수행

수행업무

CCTV 영상은 OpenWorld 데이터 수집을 위하여 부산의 자갈치, 연산, 서면 등 총 6곳에서 부산시와 협의를 통해 실제 CCTV 영상을 수집하였으며 수집된 데이터는 매일 NAS에 저장하여 영상의 흔들림, 객체 존재 유무 등 검수 기준을 수립하고 정제하여 보관하고 정제된 데이터는 크라우드소싱을 통해 레이블링 작업 및 비식별화 작업 수행
부산시에서 실제 수집된 CCTV 영상이기에 개인정보보호법으로 인하여 내부 폐쇠망 형태로 구축된 별도의 사무실을 임대하여 30명 이상의 크라우드 워커와 함께 레이블링 작업 및 검수를 수행하고 가공된 데이터를 활용한 모델 개발
데이터의 효율성 검사를 위해 해커톤 개최 및 TTA 검수 과정 진행
담당업무 : 데이터 가공툴 개발(오픈소스 커스터마이징), 크라우드워커관리, 유동인구분석 모델 개발
