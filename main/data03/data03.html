2022년 울산항 데이터 통합을 위한 플랫폼 구축 용역

과제요약

과제 기간 : 2022.12.21 ~ 2023.04.28

Github : https://github.com/zlstl1/pipeline

주요기술 : Python, Docker, Airflow, Airbyte, DBT, Oracle

주요기능 : 울산항만공사의 빅데이터 분석 플랫폼 개발 사업 수행

수행업무

PORT-MIS 기반의 기존의 데이터와 울산항 터미널, 항만 데이터의 분석을 위한 분석 및 모니터링 웹 페이지 제작
월별 선박 입항 현황, 국적/선종별 입항현황, 부두/선석별 배정량, 선석별 양·적하량, GIS 기반 데이터 등의 데이터 표출
담당업무 : 기존 레거시 데이터베이스(Oracle) 데이터 ELT 작업 수행, Airbyte를 활용하여 기존 데이터를 Datamart에 저장하고 DBT를 이용하여 변환 작업을 수행하는 일련의 과정을 Airflow를 활용하여 스케줄링
