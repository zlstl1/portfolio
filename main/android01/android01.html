"23" 이형 디스플레이 시스템 내 번역/통역 기능을 이용할 수 있는 APP 개발

과제요약

과제 기간 : 2021.04.19 - 2021.08.30

Github : https://github.com/zlstl1/LGD_Barrot

주요기술 : Android, STT, Translation

주요기능 : 23인치의 가로로 긴 막대 형태의 이형 디스플레이에 실시간 번역 기능을 탑재한 회의용 앱 개발 수행

수행업무

다른 회의용 앱과의 차별점으로는 디스플레이에 연결된 스테레오 마이크(1개의 선에 2개의 마이크가 결합된 형태)를 통해 2개의 마이크의 소리 크기를 비교하여 번역 언어를 자동으로 선택하는 기능이 있으며 발화자의 말은 STT를 통해 텍스트로 변환 후 번역
한국어, 영어, 일본어, 중국어, 러시아어, 베트남어 총 6개의 언어를 지원하며 한컴과 협력하여 회의용 전문용어에 대한 번역 앱 모델 고도화 작업 수행
회의에 녹음된 사운드(.wav) 및 속기(.txt) 파일은 회의 종료 후 저장되어 이메일로 전송할 수 있으며 사운드 파일은 FFMPEG을 통해 후처리 작업 수행
담당업무 : 구글 STT, 한컴 번역 라이브러리 외 앱 1인 개발
