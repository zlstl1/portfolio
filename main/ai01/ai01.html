GPU 병렬처리 기반 DCP 인코딩 클라우드 서비스 개발

과제요약

과제 기간 : 2018-11-01 ~ 2019-10-14

Github : https://github.com/zlstl1/dcpConvert

주요기술 : Spring Framework, OpenDCP, CUJ2K, Prometheus, FFMPEG

주요기능 : 영화사에서는 촬영된 영화(.mp4 등)를 웹을 통해 업로드하고 이를 간단한 설정값 입력을 통해 영화관 상영 포맷인 DCP로 변환하며, 변환 속도 향상을 위해 GPU 기반의 인코딩 기능을 제공

수행업무

웹 클라우드의 형태로 파일을 업/다운로드가 가능하도록 Spring을 활용하여 사용자 권한관리, 작업 로그관리 등 페이지 제작업무를 수행함. 또한 업로드된 영상을 FFMPEG을 통해 Framerate 변환, TIFF 이미지 추출 등의 영상 처리 수행
CUJ2K를 활용하여 추출된 TIFF 이미지를 GPU기반 JPEG2000으로 압축하며 DCI 기반 XYZ 색상변환 진행. Opendcp를 활용하여 압축된 JPEG2000이미지를 DCP 포맷으로 패키징하기 위한 MXF 파일로 변환하고 DCP로 패키징
서버 자원 모니터링을 위하여 Prometheus를 연결하여 자원 사용량 모니터링 페이지 제작. 변환 기능은 REST API를 통해 작업이 가능하도록 서버 구축
담당업무 : 영상 프레임 분할, MXF 변환, DCP 패키징, 웹페이지 제작

각각의 성능은 ETRI 인증을 받았으며 ①J2K 인코딩 속도 : 35.33frame/s, ②MXF 인코딩 속도 : 1649.95frame/s, ③DCP 패키징 속도 : 3.15초, ④J2K 인코딩 품질 : PSNR 36.02dB, ⑤MXF 인코딩 품질 : PSNR 41.03dB 달성
